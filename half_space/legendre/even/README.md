# Files
|File|Description|
|----|-----------|
|compute_sensor_mi.py|script to run greedy sensor selection scheme for elastic halfspace. Results are save to a separate directory|
|fit_legendre.py|script to perform load recontruction and compute data-points in the rate-distortion curve|
|legendre_sigma_y.py|script to compute the internal stress for the elastuc halfspace|
|mi_legendre.py|(legacy) Manual greedy sensor selection|
|mi_legendre_max.py| (legacy) script to run and plot greedy sensor selection scheme for elastic halfspace.|
|normalized_mi.py|Plots to validate bounds of mutual information and normalized mutual information.|
|plot_load_recon.py|script to plot examples of reconstructed load compared to the grund truth|
|plot_sensor_mi.py|plot results from greedy sensor selection|
|plot_sigma_y.py|plots the internal stress field of elastic halfspace solution from applied load generated from Legendre coefficients|
|rate_distortion.py|computes and plots rate-distortion plot compared with the ideal rate-distortion curve. Also have plot of MSE raw data|
