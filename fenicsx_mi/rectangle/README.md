# Directories
|Directory|Description|
|---------|-----------|
|mesh.py|script to mesh rectangles with different heights|
|mesh_refinementpy|script to perform mesh refinement|
|mi_calc.py|script to compute mutual information for rectangles with different heights|
|plot_mi_field.py| (legacy) plots in mutual information field on the same shape functions as the stress shape functions|
|process_run_fea.py|script to run FEA jobs to examine he affect of geometry on mutual information. Internally ``runs run_fea.py``|
|run_fea.py|script to run FEA to examine he affect of geometry on mutual information.|
